# LinkedIn Project Description

## PostgreSQL DBA Home Lab Portfolio

**Role:** PostgreSQL DBA Lab Builder  
**Dates:** In progress

I am building a hands-on PostgreSQL DBA home lab to practice real database administration concepts in a controlled learning environment. The lab uses PostgreSQL on Ubuntu Server inside Hyper-V, with pgAdmin, PowerShell, SSH, metadata queries, screenshots, ERD evidence, and structured documentation.

Current work includes PostgreSQL setup, database/schema organization, relational table design, primary and foreign keys, constraints, indexes, audit/security/reporting schemas, backup/recovery planning, and GitHub-ready portfolio documentation.

This is an in-progress learning portfolio, not a production deployment. Credentials and sensitive environment details are intentionally excluded.
