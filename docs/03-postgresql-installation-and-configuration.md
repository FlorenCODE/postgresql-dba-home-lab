# PostgreSQL Installation and Configuration

## PostgreSQL Role in the Lab

PostgreSQL is the main database platform for this DBA learning lab. It is used to practice database creation, schema organization, relational design, constraints, indexes, role awareness, evidence collection, and backup planning.

## Installation and Service Validation

The private source evidence supports that PostgreSQL was installed and validated in the lab environment. This public version summarizes that work without publishing raw command output, host details, local IP addresses, or full system inventory.

Completed or observed:

- PostgreSQL was installed on the Ubuntu Server lab VM.
- A PostgreSQL cluster was available and online.
- The lab database was created.
- pgAdmin connected successfully to the lab database.
- PostgreSQL metadata was exported for documentation.

The lab database is used for the portfolio schema design work. Role evidence was reviewed without publishing passwords.

This portfolio does not claim advanced PostgreSQL tuning or production configuration. Configuration evidence is treated conservatively. The main focus so far is installation validation, schema design, metadata review, and DBA workflow practice.
