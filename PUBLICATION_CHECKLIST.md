# Publication Checklist

## Safe To Post

- `README.md`
- Files in `docs/`
- `linkedin-project-description.md`
- `linkedin-progress-post.md`
- Selected screenshots in `screenshots/`
- High-level descriptions of the lab architecture, tools, schema design, ERD, constraints, indexes, and learning goals

## Intentionally Excluded

- Private `evidence/` folder
- Raw command output logs
- System inventory files
- Local IP addresses and private network details
- Hostnames and unnecessary machine identifiers
- Credential-related files
- Password files
- Private keys
- Environment files
- Backup archives
- Raw schema dump files
- Raw CSV exports
- Full backup/tool evidence
- Connection strings with credentials
